4. Topics, sources and hiding?

2nd May 2020 at 9:40pm
meta starthere

I have created some optional features for TiddlyBlink. Feel free to ignore them and just create notes and connect them by links. But you may find some of these features helpful:

  • I tag tiddlers about my sources (tiddlers for book chapters, dictionary articles, webpages, etc) with the tag sources. That way they show up in the sources tab in TiddlyBlink's home page, and I can go directly to a list of them there.
  • Hiding: I like to create long outlines and table of contents tiddlers. But in the simple transclusion view of the references, these are an eyesore. So I created a tag hide to keep them from appearing in the reference section below the tiddlers. You can use this tag to hide any tiddler you don't want to appear there.
    • You can also use the tag outlines for the same effect (left this in for backwards compatibility).
  • Notes are tiddlers created with the clock icon buttons. They have a date and time stamp as a title. They are for when you want to create a note but don't want to waste time thinking up a title.
  • Topics are all the other tiddlers. To be precise, topics are any tiddler not tagged sources or outlines, and that don't have 202 in the tiddler title.
  • Meta tiddlers are the tiddlers for documentation of TiddlyBlink. They are tagged meta.
